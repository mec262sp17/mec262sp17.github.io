Linear regression¶

Objective is to fit a model of type \( y = mx + c \) to noisy data.

  • Useful in predicting continuous value outputs
  • Example: Predicting steering angle for self-driving car, height of a person from age and weight, etc.

Linear SVM¶

Objective is to find a plane that best separates a given set of points.

  • Useful in classifying a given data set into different groups
  • Example: Identifying sentiment of a twitter post, identify emotional expression from image, look at scans and classify them into cancerous or non-cancerous.

Linear methods for nonlinear fitting¶

  1. Convert nonlinear fitting problem to linear task.
  2. Apply linear fitting methods.

A = np.vstack((-2*x,-2*y,np.ones(len(x)))).T
b = -(x**2 + y**2)

ATA_inv = np.linalg.inv(np.dot(A.T,A))
Atb = np.dot(A.T,b)
W = np.dot(ATA_inv, Atb)

x_c = W[0]
y_c = W[1]
r = np.sqrt( x_c**2+y_c**2 - W[2])

Limitations of previous methods¶

  • Need to choose weights, nonlinear function, etc manually.
  • Do not generalize well to complex data distributions

Neural networks¶

  • Needn't worry about specifying the exact type of nonlinearity.
  • Combine multiple simple models to make complex predictors

Simple neuron compute¶

  • Dendrites combine incoming signals from different neurons
  • Combine them and transmit to synapses via axons
  • Axons are connected to dendrites of other neurons

Neuron model¶

  • Combine large number of simple compute units to achieve complex behavior.

Neural network model¶

  • Connect multiple simple compute units to form a complex circuit
  • Different architectures for different applications
  • Deep learning for image/audio/language processing
  • Generative networks for producing new data, medicine, drugs, etc.
